## Comandos Rápidos

1. Iniciar servidor na porta 3002
```bash
npx -y serve -l 3002
```

2. Abrir editor no navegador
- URL: http://localhost:3002/editor.html

3. Abrir dashboard no navegador
- URL: http://localhost:3002/app.html

## Testes a Executar

### Cores
- Clicar em cada preset de cor sólida
- Verificar se preview atualiza
- Clicar em gradientes
- Testar color picker avançado

### Fundos
- Clicar em cada cor de fundo
- Upload de imagem de fundo
- Remover imagem de fundo

### Banner
- Ir para aba Links
- Preencher Banner de Destaque
- Ativar toggle
- Verificar no preview

### Footer/Assinatura
- Ir para aba Perfil
- Preencher Sua Assinatura
- Verificar no preview

### Salvar/Carregar
- Criar novo projeto
- Editar dados
- Voltar ao dashboard
- Reabrir projeto
- Verificar se dados persistiram
