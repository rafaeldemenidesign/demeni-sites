# Demeni Sites - Plataforma de Criação de Sites

Plataforma SaaS para criação de sites link-na-bio com sistema de níveis e gamificação.

## Funcionalidades

- ✅ Editor visual de sites
- ✅ Sistema de níveis/XP com bordas
- ✅ Autenticação de usuários
- ✅ Gerenciamento de projetos
- ✅ Banners, vídeos, links personalizados

## Tecnologias

- HTML5, CSS3, JavaScript
- localStorage (MVP)
- Supabase (backend futuro)

## Desenvolvimento

```bash
npx serve -l 3002
```

## Estrutura

```
demeni-sites/
├── app.html         # Dashboard
├── editor.html      # Editor de sites
├── js/
│   ├── auth.js      # Autenticação
│   ├── userData.js  # Dados do usuário
│   ├── xpSystem.js  # Sistema de XP/níveis
│   ├── dashboard.js # Dashboard
│   └── editor.js    # Editor
└── css/
    └── app.css      # Estilos
```
