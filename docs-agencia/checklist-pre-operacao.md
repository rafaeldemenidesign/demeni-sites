# Checklist Pré-Operação — Demeni Sites

> Tudo que precisa estar pronto antes de começar a vender com vendedor externo.
> Última atualização: 02/03/2026

---

## 🔴 Fase 1 — Sem Isso Não Vende (Bloqueante)

### 1. Link de Pagamento
- [x] Criar conta comercial (Mercado Pago) ✅
- [x] Configurar link de pagamento para **D-1 Cartão Digital:**
  - PIX: **R$ 149,00** (preço líquido)
  - Cartão: **R$ 174,00** parcelado vendedor em até 3x s/ juros (parcela ~R$ 58)
  - Link: **https://mpago.li/2kh8AFG**
- [x] Configurar link de pagamento para **D-2 Minisite Profissional:**
  - PIX: **R$ 250,00** (preço líquido)
  - Cartão: **R$ 287,00** parcelado vendedor em até 5x s/ juros
  - Link: **https://mpago.li/2NHMQ8y**
- [ ] Testar Pix e Cartão de Crédito com compra real
- [ ] Gerar QR Code de pagamento (para vendedor mostrar na hora)

**Estratégia:** PIX = preço real que você recebe líquido. Cartão = preço inflado para cobrir as taxas do Mercado Pago. Parcelamento 5x absorvido pelo vendedor. Redirecionamento pós-pagamento: core.rafaeldemeni.com

### 2. Chaveiros NFC Físicos
- [ ] Produzir lote mínimo de **10 unidades de demo** (para o vendedor usar na demonstração)
- [ ] Produzir **lote inicial de entrega** (pelo menos 30 unidades para os primeiros clientes)
- [ ] Definir fornecedor/processo de produção artesanal
- [ ] Definir custo unitário e margem
- [ ] Programar NFC com URL de exemplo para demo

**Por que é bloqueante:** O chaveiro NFC é o diferencial que fecha a venda. Sem demo físico = venda genérica de site = muito mais difícil.

### 3. Vídeo de Vendas
- [ ] Gravar vídeo para a **Landing Page** (sites.rafaeldemeni.com)
- [ ] Gravar vídeo para a **Página do Vendedor** (sites.rafaeldemeni.com/vendedor-5k)
- [ ] Subir vídeos no YouTube (não listado)
- [ ] Substituir placeholders pelos embeds nas páginas
- [ ] Thumbnail já está pronta ✅

**Por que é bloqueante:** A landing page e a página de recrutamento dependem do vídeo para converter. Placeholder = amadorismo.

### 4. Portfólio de Sites Demo
- [ ] Criar **3-5 sites D-2** com negócios fictícios mas realistas (ex: restaurante, salão, personal trainer, loja, clínica)
- [ ] Publicar cada um com URL acessível (ex: s/restaurante-demo)
- [ ] Montar galeria no celular do vendedor para demonstração rápida

**Por que é bloqueante:** O vendedor precisa mostrar exemplos reais. "Confia, vai ficar bonito" não vende.

## 🟡 Fase 2 — Operar com Vendedor (Importante)

### 5. Script de Vendas
- [ ] Escrever roteiro de abordagem (como chegar no comerciante)
- [ ] Listar 5 objeções mais comuns + respostas prontas:
  - "Já tenho Instagram" → resposta
  - "Tá caro" → downsell pro D-1
  - "Não preciso de site" → argumento do Google
  - "Vou pensar" → urgência
  - "Já tenho site" → mostrar diferencial NFC
- [ ] Definir pitch de 30 segundos (elevator pitch)
- [ ] Definir pitch completo (2-3 minutos com demo NFC)
- [ ] Treinar vendedor com o script

### 6. Contrato / Termos do Vendedor
- [ ] Definir tipo de vínculo (freelancer/autônomo, sem CLT)
- [ ] Redigir termos simples:
  - Comissão proporcional à média diária
  - 1/dia = R$1.000, 2/dia = R$2.000 ... 5/dia = R$5.000
  - Pagamento mensal (dia X)
  - Sem exclusividade
  - Uso de marca autorizado durante contrato
- [ ] Definir período de experiência (ex: 30 dias)
- [ ] Validar com advogado (opcional mas recomendado)

### 7. Fluxo de Onboarding do Cliente
- [ ] Definir o que o vendedor coleta na hora da venda:
  - Nome do negócio
  - Tipo de serviço/produto
  - Telefone/WhatsApp
  - Logo (se tiver)
  - Fotos (se tiver)
  - Redes sociais
- [ ] Criar formulário simples (Google Forms ou WhatsApp estruturado)
- [ ] Definir prazo de entrega (48h prometido — é viável com 8 sites/dia?)
- [ ] Definir quem avisa o cliente quando o site está pronto
- [ ] Definir como entrega o chaveiro NFC (junto com a venda? Enviado depois?)

### 8. WhatsApp Business
- [ ] Configurar WhatsApp Business com número comercial
- [ ] Criar mensagem de boas-vindas automática
- [ ] Criar catálogo com D-1 e D-2
- [ ] Criar etiquetas (Lead, Em produção, Entregue, Pós-venda)
- [ ] Criar mensagem rápida pós-venda

## ⚪ Fase 3 — Escalar (Quando Tiver Caixa)

### 9. Tráfego Pago (Meta Ads)
- [ ] Criar conta de anúncios no Meta Business
- [ ] Instalar Pixel na landing page
- [ ] Criar 3-4 criativos de teste
- [ ] Definir público-alvo (donos de pequenos negócios, 25-55 anos, local)
- [ ] Investir R$ 5.000/mês (meta: mês 2-3)

### 10. Automações Pós-Venda
- [ ] Email/WhatsApp automático: "Seu site está pronto!"
- [ ] Pesquisa de satisfação 7 dias após entrega
- [ ] Oferta de upgrade D-1 → D-2 (30 dias após compra)
- [ ] Programa de indicação (cliente indica, ganha desconto)

### 11. Segundo Vendedor
- [ ] Quando demanda > capacidade de produção
- [ ] Usar mesmo script validado pelo primeiro vendedor
- [ ] Outra cidade/região para não competir
